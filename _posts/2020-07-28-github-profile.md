记录一下如何配置GitHub Profile，在个人页中显示详细信息。

<!-- more -->

## 创建GitHub Profile

1. 新建仓库，仓库名与用户名一致；
2. 选择仓库属性为`Public`；
3. 勾选README初始化选项；
4. 仓库中`README.md`文件中的内容会显示为GitHub Profile；
