## GitHub Code Vault

```text
寄蜉蝣于天地，渺沧海之一粟。哀吾生之须臾，羡长江之无穷。挟飞仙以遨游，抱明月而长终。知不可乎骤得，托遗响于悲风。——《赤壁赋》
```

1. GitHub Code Vault（GitHub代码保险库）是由GitHub Archive Program（GitHub代码永久保存计划）设立的代码档案库，旨在保存开源软件以供未来使用；
2. 2020年2月2日，GitHub对于所有公共仓库进行快照存档，选取其中满足以下三个条件的仓库加入代码档案库：

    - 前80天内有提交更改的所有仓库；
    - 前365天内有提交更改的所有仓库（至少有一颗星）；
    - 250颗星及以上的所有仓库，无需理会其最后更新日期为何时；
